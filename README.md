## ✨ Features

| Feature | Description |
|---|---|
| 🌗 **Adaptive Theming** | Full dark & light mode support via `isSystemInDarkTheme()` |
| 🌀 **Animated Background** | GPU-friendly canvas blob/mesh animation (8s loop) |
| 💀 **Shimmer Skeleton** | Smooth skeleton loader while ad fetches |
| ⏱️ **Countdown → Skip** | Circular countdown morphs into a skip button |
| ✨ **Gradient CTA** | Animated shine sweep on the call-to-action button |
| 🌟 **Star Rating Bar** | Native AdMob star rating rendered in Compose |
| 📦 **Lifecycle-Safe** | Auto-destroys `NativeAd` on `DisposableEffect` |
| 🔥 **Spring Physics** | Bouncy spring animations on all interactive elements |
| 📐 **Edge-to-Edge** | Status bar aware with `statusBarsPadding()` |
